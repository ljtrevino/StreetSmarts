Future Work

Since we only had around 4 weeks to complete this project, there are a number of ideas and features that we would like to implement in order to further improve upon our project. Here we highlight a few of these features that would make our project even more robust and applicable in the real world:

Additional Emergency Types

A feature we would have liked to implement, if we had more time, was support for other emergencies. In a real life system, we would have police stations and hospitals included in the sources, and depending on the emergency, different responders would be called. The lamps would be able to fully utilize their RGB capability and change color based on the nature of the emergency.

More Lamps

A real system implemented in a city like Cambridge would involve a much larger quantity of street lamps. It is estimated that the US alone has 26 million street lamps, much more than the 31 that we implemented. It would be very interesting to see how a system like this scales to a larger neighborhood, town, or even a metro area.

More Efficient API Use

Since the Google Distance Matrix API was such a pain point, it would be very interesting to see if we could create a more efficient use. For example, configuring the server to automatically query the API every minute or so, then running the Dijkstra algorithm on the most updated data, rather than just querying the API every time there is a request. This would also lead to much improved pathfinding speeds, and every second matters with emergencies. This wasn’t something we could implement on the 6.08 server since the code only runs when receiving a request, but it would be fun to set up.

RGB Lighting in Demo Map

Our mini map currently has a red LED to represent the fire station, but the rest of the LEDs are purely white. In continuation of the “additional emergency types” expansion, the ability to address all of our LEDs and control their color would be something interesting and cool. With our limited time, figuring this out would have been more pain than it would be worth, but as a future project it would be fun. This would also allow us to not have the fire station location hardwired in with the red LED, and allow us to add many other cool additional features.
